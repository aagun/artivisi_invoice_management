# Aplikasi Invoice Management

Aplikasi ini dipakai untuk mengelola invoice dan menyambungkan dengan berbagain metode pembayaran.
Diantara metode pembayaran yang akan disupport antara lain:

* Virtual Account
  * Bank BNI
  * Bank CIMB
  * BSI
  

* e-Wallet
  * GoPay
  * Dana
  * OVO


* QR Payment
  * QRIS
